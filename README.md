clu-node-red

### About

These are my automation flows for Home Assistant with Node Red.

#### Flows

Family Bell; my kid's morning automations
Thermostats; rudimentary thermostat control (WIP)
Lights / Switches; control of all lights, switches, outlets, etc
Presence; Just door monitoring
Allergy; Allergy info gathering for a HA dashboard for my son
Maintenance; sensor battery maintence, medication reminders, HELOC APR monitoring, etc
